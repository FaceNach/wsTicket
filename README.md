# README

## English

### Ticket Queue System with WebSocket

This is a real-time ticket queue management system built with WebSocket technology. The project consists of a Bun-based WebSocket server and a React frontend application that enables users to create, track, and manage service tickets in real-time.

**Technologies Used:**
- **Backend**: Bun runtime, WebSocket API, TypeScript, Zod for validation
- **Frontend**: React 19, TypeScript, Vite, React Router, Tailwind CSS

The system allows multiple clients to connect simultaneously and receive real-time updates when tickets are created or assigned to service desks.

## Español

### Sistema de Colas de Tickets con WebSocket

Este es un sistema de gestión de colas de tickets en tiempo real construido con tecnología WebSocket. El proyecto consiste en un servidor WebSocket basado en Bun y una aplicación frontend en React que permite a los usuarios crear, rastrear y gestionar tickets de servicio en tiempo real.

**Tecnologías Utilizadas:**
- **Backend**: Runtime Bun, API WebSocket, TypeScript, Zod para validación
- **Frontend**: React 19, TypeScript, Vite, React Router, Tailwind CSS

El sistema permite que múltiples clientes se conecten simultáneamente y reciban actualizaciones en tiempo real cuando se crean o asignan tickets a las ventanas de servicio.
