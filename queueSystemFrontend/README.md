# Queue System Frontend

A React-based queue management system frontend with real-time WebSocket communication. Features include a kiosk for ticket issuance, a display board for showing current tickets, and operator desks for ticket management.

## Features

- Real-time ticket queue management via WebSocket
- Three main interfaces: Kiosk (ticket request), Board (display), Desk (operator)
- Support for normal and preferential ticket requests
- Reactive UI with Tailwind CSS
- Responsive design for various display sizes

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM

# Sistema de Colas - Frontend

Un frontend basado en React para un sistema de gestión de colas y turnos con comunicación WebSocket en tiempo real. Incluye una kiosko para emitir turnos, un panel de visualización para mostrar los turnos activos, y escritorios de operador para la atención.

## Características

- Gestión de cola de turnos en tiempo real vía WebSocket
- Tres interfaces principales: Kiosko (solicitud), Panel (visualización), Escritorio (operador)
- Soporte para turnos normales y preferenciales
- Interfaz reactiva con Tailwind CSS
- Diseño responsivo para distintos tamaños de pantalla

## Stack Tecnológico

- React 19
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM
